先點選 Create Contract

把基本資料輸入進去
![Step 2](screenshots/step2.png)

錢包批准完後就會部署上去
![Step 3](screenshots/step3.png)

透過 Manifold Studio 也可以直接 mint
![Mint](screenshots/mint.png)

也可以設定版稅分潤
